# Epic Plan: Rebranding and UX/UI Modernization

**Status:** Not Started

## 1. Goals
- Fully rebrand the application from "CoughDrop" to "LingoLinq AAC".
- Modernize the user interface (UI) with a new color scheme, typography, and layout.
- Improve the user experience (UX) by simplifying complex workflows and improving accessibility.

## 2. Scope
- **Logos & Branding:** Replace all existing logos and favicons.
- **Color Palette:** Implement a new, modern, and accessible color scheme across the entire application (Rails and Ember).
- **Typography:** Standardize fonts for consistency.
- **Component Refactoring:** Update major UI components (e.g., navbars, buttons, modals) to the new design system.
- **Layouts:** Refactor key page layouts for better information hierarchy.

## 3. Key Tasks (Initial)
- [ ] Inventory all existing brand assets.
- [ ] Replace all instances of the name "CoughDrop" in user-facing views.
- [ ] Implement the new primary color palette.
- [ ] Update the main application layout and navigation bar.

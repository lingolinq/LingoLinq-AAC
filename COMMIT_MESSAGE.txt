**Migration Pattern Applied:**
1. Added `import { inject as service } from '@ember/service';`
2. Added service injections: `stashes: service('stashes')`, `persistence: service('persistence')`, `appState: service('app-state')` as needed
3. Replaced utility imports with service injections
4. Replaced `stashes.get(...)` with `this.stashes.get(...)` or `_this.stashes.get(...)` in callbacks
5. Replaced `persistence.get(...)` with `this.persistence.get(...)` or `_this.persistence.get(...)` in callbacks
6. Replaced `app_state.get(...)` with `this.appState.get(...)` or `_this.appState.get(...)` in callbacks
7. Fixed `this` context issues in callbacks by using `_this` pattern
8. Removed unused imports where services were not actually used
9. Added computed properties and aliases for template compatibility

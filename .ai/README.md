# AI Development Tools 🤖

This directory contains active AI development tools and configurations for LingoLinq AAC development.

## Directory Structure

### 📁 agents/
AI assistant configurations and coordination:
- `claude/config.json` - Claude AI assistant configuration
- `gemini/config.json` - Gemini AI assistant configuration
- `coordinator/` - Multi-agent coordination system
- `project-constraints.json` - Shared development constraints
- `README.md` - Agent setup instructions

### 📁 context/
AI development context and project analysis:
- Architecture mappings and component analysis
- Project state tracking and session management
- Technical audits and implementation roadmaps
- Team workflow coordination files

### 📁 tools/
Development utilities and scripts:
- `deepwiki-mcp/` - DeepWiki MCP server for project context
- `setup-dev-environment.sh` - Development environment setup
- `verify-environment.sh` - Environment verification
- `compile-docs.ps1` - Documentation compilation
- `core_ngrams.rb` - Core language processing utilities
- `i18n_generator.rb` - Internationalization tools
- `launch-agentic.ps1` - AI agent coordination launcher

## Quick Start

1. **Agent Setup**: Check `agents/` for AI assistant configurations
2. **Context Loading**: Use `context/` files for project understanding  
3. **Development Tools**: Run scripts from `tools/` for workflows

## Key Files

- `agents/project-constraints.json` - Development constraints and version locks
- `tools/setup-dev-environment.sh` - One-command environment setup
- `context/README.md` - Current project architecture overview

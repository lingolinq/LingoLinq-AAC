## Top 5 MVP Blockers

### 1. Critical Authentication & Authorization Vulnerabilities
**Severity:** CRITICAL | **Domains:** Dependencies, API, Rails
**Fix Effort:** 1-2 weeks

ruby-saml has CRITICAL auth bypass CVEs (CVE-2025-25291, CVE-2025-25292) allowing forged SAML assertions. Combined with `permit_all_parameters=true` globally disabling Strong Parameters, an attacker can escalate privileges and access all user data. The purchasing webhook is completely unauthenticated.

**Impact:** Complete system compromise. Any school/hospital deployment fails security review immediately.

### 2. SQL Injection in Production Code
**Severity:** CRITICAL | **Domain:** Rails Backend
**Fix Effort:** 1-2 weeks

String interpolation in SQL queries confirmed in 16 files including `BoardDownstreamButtonSet.reconcile`, `LogSession`, `User`, `Organization`, and multiple controllers.

**Impact:** Full database compromise including all user PII and communication logs.

### 3. No CI/CD Pipeline or Automated Testing
**Severity:** HIGH | **Domains:** Infrastructure, Ember
**Fix Effort:** 1 week basic, ongoing for coverage

Zero GitHub Actions workflows. 3 frontend tests across 102 test files. No automated security scanning. render.yaml deploys web from 'functional-baseline' but worker from 'main'.

**Impact:** Cannot safely deploy fixes. Every deployment is unvalidated.

### 4. Missing COPPA/FERPA Compliance for Target Market
**Severity:** HIGH | **Domain:** Privacy/Compliance
**Fix Effort:** 2-4 weeks

No COPPA age verification. No FERPA educational records logging. BAAs with Render/AWS unconfirmed. Zendesk receives PII without BAA.

**Impact:** Schools cannot legally deploy (FERPA/COPPA). Hospitals cannot deploy (HIPAA).

### 5. SSL Verification Disabled Across Production Code
**Severity:** HIGH | **Domains:** Infrastructure, Rails, Privacy
**Fix Effort:** 2-3 days

`ssl_verifypeer: false` in 8+ production locations. No Content Security Policy. No security headers.

**Impact:** Fails HIPAA technical safeguard requirements. User communication data could be intercepted.
